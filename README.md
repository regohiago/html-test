# Página HTML — Tags Básicas

## 📌 Descrição
Projeto simples desenvolvido para praticar a utilização de tags HTML fundamentais e estruturação de conteúdo.

## 🎯 Objetivo
Aplicar conceitos básicos de HTML, incluindo:

- Hierarquia de títulos
- Parágrafos
- Formatação de texto
- Listas
- Links
- Tags semânticas

## 🛠 Tecnologias
- HTML5

## 🧩 Conceitos Praticados
- Headings (`h1` → `h6`)
- Parágrafos (`p`)
- Ênfase (`strong`, `mark`, `i`, `u`, `small`)
- Listas (`ul`, `ol`, `li`)
- Links (`a`)
- Elementos adicionais (`hr`, `blockquote`, `abbr`, `sub`, `sup`)
- Tags legadas para estudo (`font`, `del`)

## ⚠ Observações
Projeto educacional.  
Sem uso de CSS ou JavaScript.  
A tag `<font>` foi utilizada apenas para fins didáticos (obsoleta em HTML5).

## ✅ Resultado
Página estática demonstrando visualmente o uso das principais tags HTML.
